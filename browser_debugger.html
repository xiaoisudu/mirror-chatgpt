💡 使用说明

这个工具帮助你理解 React Flight Protocol 的全局对象表结构。


在浏览器控制台中使用：

// 1. 查看原始数据流
console.log(window.__reactRouterContext);

// 2. 如果数据已加载到 globalTable 变量
console.log('全局表:', globalTable);

// 3. 查找特定索引
console.log('索引 43:', globalTable[43]);

// 4. 解析引用示例
function resolve(idx) {
    return globalTable[idx];
}
console.log('解析 _43 =', resolve(43));
